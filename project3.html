Note It App

UX & UI case study - Individual Project - Autumn 2022

Intro

From all of the apps that come pre-installed on iOS, Reminders seems to be the app that people forget about. If they're writing a list of some sort, they'll write it in Notes or Pages, If they need to be reminded about something, an appointment is set into Calendar, Reminders is actually a very powerful tool that is unfairly (or fairly) ignored, It's a seemingly simple app, but it lacks an intuitive design, and takes time to acknowledge it's great features.

The Goal

The main purpose is to create an app that would resemble Reminders' great features. With improvements that will push these functions further to help you more intelligently manage your agenda and all of the tasks you need to complete. For a better user experience the most attractive features should be simplicity, and accessibility.

Creating personas

After a research was made, in order to understand & meet user's needs, personas with different backgrounds and experiences were created.

User Flow

The path a user follows through the app interface to complete a task. In this case the task is to create and edit a note.
